git切换远程分支命令

发表于 2018-07-19 | 分类于 git

1.克隆项目 

git clone <项目链接>

2.进入项目 

cd work

3.列出所有分支 

git branch -a

输出 

remotes/origin/dev
remotes/origin/release

4.切换远程开发分支 

git checkout -b dev origin/dev

切换远程分支origin/dev

5.切换远程正式分支 

git checkout -b release origin/release

6.切换回本地dev分支，开始开发 

git checkout dev
