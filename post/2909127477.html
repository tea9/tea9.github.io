####接口名：getLocation
####描述：获取门店信息(未登录状态默认为2号门店)

|参数 |类型| 说明|
|—-|—–|
|address |string|用户的地址信息 地址详情 data.detail|
####示例：


socket.emit("getLocation", {
        "address": "汇龙北纬"
    });

####监听到的json：

参数 说明
data 门店id
####示例： 

{
    "status":1,
    "data":3
}

####接口名：login
####描述：登录接口

|参数|类型|说明|
|——–|——–|
|username|string|用户名|
|password|string|用户密码|
|uuid|string|设备id|

####示例：

var data = {
    "password": ‘pwd’,
    "username": ‘username’,
    "uuid": ‘860150151509603’
}
socket.emit('login', data);

####返回的json

参数 类型 说明
token string 返回的token 用于获取用户信息

####示例：

{
"status":1,
"data":{
    "token":"6vbnhxt2uFo5giTlQz6LwT8TcbqtJDsZ2AvtJEx1rl1ub82yCZ1WClFS1SM7lX8d"
    }
}

####接口名：register
####描述：注册

|参数|类型|描述|
|—-|—–|
|username|string|用户名|
|password|string|密码|
|phone|string|手机号|
|captcha|string|手机验证码|
|email|string|邮箱地址|

####示例：


    socket.emit('register',  {
        "password": 'pwd',
        "username": 'username',
        "phone":'13578455745',
        "captcha":'7845',
        "email":'123@qq.com'
    });

####返回的json：

参数 类型 描述
status int 状态码成功返回1
data string 成功或者失败的描述 

{
    "status":1,
    "data":"Registration successful"
}

####接口名： changePassword
####描述：修改密码接口

|—–|——|
|oldPassword|string|旧密码|

    socket.emit('changePassword', {
        oldPassword: '123,
        password: '456'
    });

####返回的json

|参数|类型|描述|
|—-|—-|
|status|int|状态修改成功状态为1修改失败为0|
|data|string|描述信息|
####示例：


    {
        "status":1,
        "data":"password changed ok."
    }

####接口名：search
####描述：根据关键字查询商品

|参数|类型|描述|
|—-|—-|
|keyword|string|关键字|
|start|int|每页查询的起始位置|
|offset|int|每页查询的条数|
|areaID|int|门店id|

####示例：


    socket.emit('search', {
        'keyword': '水',
        'start': 0,
        'offset': 10,
        'areaID':2
    });

####返回的json;

|参数|类型|描述|
|—-|—-|
|id|arr|id数组 把id拼接成数组调用getProductList接口|

####接口名：getProductList
####描述：根据id数组获取商品详情

|参数|类型|描述|
|—-|—-|
|idList|arr|id数组 |
|areaID|int|门店id|

####示例：


    socket.emit('getProductList', {
        'idList': ’37,38,767,4142,4142,4141,4141,4141,4141,664‘,
        'areaID':2
    });

####返回的json

|参数|类型|描述|
|——|——-|
|data|arr|返回的商品数组|
|id|int|商品id|
|productName|string|商品名称|
|productImage|string|商品图片路径|
|price|double|价格|
|isGoods|bool|进口 1为true 0为false|
|isHot|bool|热卖 1为true 0为false|
|isNew|bool|新品 1为true 0为false|
|isOnsaled|bool|促销 1为true 0为false|
|isRecommanded|bool|推荐 1为true 0为false|
|amounts|int|库存数量|
|sales|string|月销售数量|
|out|bool|判断是否有库存 0为false 1为true|
