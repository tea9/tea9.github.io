unicorn在安卓中使用

Author: tea9 Date: 二月 19, 2021  16:30:27 Category: Security

https://github.com/ElderDrivers/EdXposed

前提先安装magisk

下载

v23.5

https://github.com/RikkaApps/Riru/releases

v21.1

https://github.com/yangzhaofeng/riru-core

adb push magisk-riru-v21.1.zip /sdcard/

magisk manager

选择模块

从本地安装

选择magisk-riru-v21.1.zip

安装成功后重启

下载EdXposed-SandHook-v0.4.6.2.4529.-release.zip

https://github.com/ElderDrivers/EdXposed/releases/tag/v0.4.6.4

adb push EdXposed-SandHook-v0.4.6.2.4529.-release.zip /sdcard

magisk manager

选择模块

从本地安装

选择EdXposed-SandHook-v0.4.6.2.4529.-release.zip

安装成功后重启

下载edxoosed manager

https://github.com/ElderDrivers/EdXposedManager/releases

adb install EdXposedManager-4.5.7-45700-org.meowcat.edxposed.manager-release.apk

下载justtrustme

JustTrustMe：屏蔽App SSL Pinning校验利器

https://files-cdn.cnblogs.com/files/pingming/JustTrustMe.apk

安装

adb install JustTrustMe.apk

edxposed模块中开启 重启手机
