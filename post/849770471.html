使用pyenv切换python版本

Author: tea9 Date: 六月 12, 2018  0:00:00 Category: Code

关键代码

which python3.6

brew install pyenv

pyenv install --list // 可安装的python版本
pyenv install -v 2.7.10 // 安装python
pyenv rehash // 更新
pyenv versions // 查看已经安装的python版本
pyenv global 2.7.10 // 切换python版本
pyenv global system // 切换原来版本

python --version
