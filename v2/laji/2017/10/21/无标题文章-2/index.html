####接口名：register ####描述：注册

参数 类型 描述
username string 用户名
password string 密码
phone string 手机号
captcha string 手机验证码
email string 邮箱地址

####示例：


	socket.emit('register',  {
		"password": 'pwd',
		"username": 'username',
		"phone":'13578455745',
		"captcha":'7845',
		"email":'123@qq.com'
	});

####返回的json：

参数 类型 描述
status int 状态码成功返回1
data string 成功或者失败的描述 

{
    "status":1,
    "data":"Registration successful"
}

		'idList': ’37,38,767,4142,4142,4141,4141,4141,4141,664‘,
		'areaID':2
	});
